DOM Widgets

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects. That way, programming languages can connect to the page. A Web page is a document. The project goes on and require us to use factory functions and then connect that functions to the DOM Widgets.
